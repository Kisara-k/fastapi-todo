Create and activate venv

```bash
python -m venv venv
# On Windows
venv\Scripts\activate
# On macOS/Linux
source venv/bin/activate
```
(To exit venv)
```bash
deactivate
```

Refer to the [FastAPI User Guide](https://fastapi.tiangolo.com/tutorial/)

```bash
pip install fastapi
pip install "uvicorn[standard]"
```
